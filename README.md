# Customer Behavior Analysis Dashboard

A complete end-to-end data analytics project focused on understanding customer shopping behavior using SQL, Python, and Power BI.

## Project Overview

This project analyzes customer purchasing patterns, subscription behavior, product performance, discounts, ratings, and revenue trends.

The workflow includes:

- Data Cleaning & Exploration using Python
- Business Problem Solving using SQL
- Interactive Dashboard Creation using Power BI

## Tools Used

- Python
- Pandas
- SQL (MySQL)
- Power BI
- Jupyter Notebook

## Key Insights

- Compared subscriber vs non-subscriber spending behavior
- Identified top revenue-generating categories
- Analyzed discount usage patterns
- Found high-rated products
- Explored customer segmentation based on previous purchases
- Visualized regional purchasing trends

## Dashboard Features

- KPI Cards
- Revenue Analysis
- Product Performance
- Customer Segmentation
- Subscription Insights
- Interactive Filters
